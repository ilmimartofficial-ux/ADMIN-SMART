📊 Sumber Data

Sistem membandingkan harga retail antara data IPOS5 (Grosir) dan IPOS4 (Mart) untuk setiap item yang sama (berdasarkan Kode atau Nama).

🔄 Mode Sinkronasi
IPOS5 → IPOS4: Harga IPOS5 jadi acuan
IPOS4 → IPOS5: Harga IPOS4 jadi acuan
Harga Tertinggi: Ambil harga lebih tinggi
Harga Terendah: Ambil harga lebih rendah
⚠ Perhatian

Sinkronasi akan merekam riwayat perubahan harga. Pastikan data sudah di-import terlebih dahulu sebelum sinkronasi.

📖 Panduan & Cara Deploy
🧠 Smart Column Mapper — Cara Kerja
✓ Auto-Deteksi

Sistem scan 20 baris pertama file Excel untuk menemukan baris header. Kemudian cocokkan nama kolom ke field yang dibutuhkan menggunakan keyword matching.

🔍 Fuzzy Match

Kolom "Harga Pokok", "Modal", "H. Pokok", "Cost" → semua dikenali sebagai kolom Modal. Variasi nama kolom apapun selama mengandung keyword yang relevan.

👆 Manual Override

Jika auto-deteksi kurang tepat, tampil modal pemetaan kolom. Anda bisa pilih kolom yang benar dari dropdown untuk setiap field secara manual.

📋 Preview Data

Sebelum import, lihat preview 5 baris pertama data yang akan diimport. Pastikan kolom sudah benar, lalu klik "Konfirmasi & Import".

🔄 Sinkronasi Harga — Cara Pakai
1️⃣ Import Data

Import dulu file IPOS5 (Grosir) dan DATA_IP4 (Mart) melalui menu "Update Data Excel". Pastikan keduanya sudah terimport dengan benar.

2️⃣ Cek Perbedaan

Buka menu "Sinkronasi Harga" di sidebar. Sistem otomatis mendeteksi item mana yang harga retailnya berbeda antara IPOS5 dan IPOS4.

3️⃣ Pilih Mode

Pilih mode sinkronasi: IPOS5→IPOS4, IPOS4→IPOS5, atau ambil harga tertinggi/terendah. Centang item yang ingin disinkronisasi.

4️⃣ Terapkan

Klik "Terapkan Sinkronasi". Perubahan harga akan dicatat di Riwayat Harga dan data tersimpan otomatis.

🚀 Cara Deploy Online (Gratis)
🟢 Netlify (Paling Mudah)
  1. Buka netlify.com → Sign Up gratis
  2. Drag & drop file HTML ke halaman Netlify
  3. Dapat URL: xyz.netlify.app
  4. Akses dari HP, tablet, komputer mana saja
🔵 GitHub Pages
  1. Buat akun GitHub → New repository
  2. Upload file HTML → rename jadi index.html
  3. Settings → Pages → Deploy from main
  4. URL: username.github.io/nama-repo
📁 cPanel Hosting
  1. Login cPanel → File Manager
  2. Buka folder public_html
  3. Upload file → rename jadi index.html
  4. Akses via domain/subdomain Anda
